大先生バトル - 職場の理不尽と戦うカードゲーム

📋 アプリ概要

**「大先生バトル」**は、職場の厄介な上司や先輩（通称：大先生）との不毛な会議をシミュレートした、ブラウザで遊べるターン制カードバトルRPGです。

プレイヤーは若手・中堅社員となり、意味のない横文字や責任転嫁を繰り返す「大先生」を論破し、無事に定時退社（勝利）することを目指します。

🧑‍💼 キャラクター紹介

敵：大先生 (Dai-Sensei)

オフィスに生息する、少し不思議で愛すべき（？）困った存在。

特徴: 自信満々だが準備不足。「ちょうどやろうと思ってた」が口癖。

攻撃方法: 話が長い、論点をずらす、曖昧な横文字、責任転嫁。

HP: 「プライド」。これを0にすると撃退できます。

弱点: ファクト（事実）、議事録、具体的な期限設定。

プレイヤー：あなた

プロジェクトを何とか進めようと奮闘する社員。

HP: 「SAN値（メンタル）」。大先生の長話や理不尽な攻撃を受けると減っていき、0になると残業確定（敗北）です。

🎮 遊び方

ゲーム開始: 「ミーティング開始」ボタンを押してバトルスタート。

自分のターン: 手札のカードから1つ選び、大先生に行動を起こします。

攻撃系カード：大先生のプライドにダメージを与えます。

回復系カード：自分のSAN値を回復します。

相手のターン: 大先生が理不尽な言動（攻撃）をしてきます。

勝敗:

WIN: 大先生のプライドを削り切る → 定時退社 🍻

LOSE: 自分のSAN値が尽きる → 残業確定（朝までコース） 😵

❌ Yes/Noで質問

攻撃

「結論からお願いします」と迫ります。話を煙に巻きたい大先生の逃げ道を塞ぐ強力な一手。

⏱️ 期限の設定

攻撃

「具体的にいつですか？」と詰め寄ります。「なる早で」等の曖昧な指示を許しません。

🛡️ 事実確認

大攻撃

「その情報ソースは？」とエビデンスを求めます。聞きかじりの知識で話す大先生へのクリティカル攻撃。

☕ スルー

回復

相手の話を華麗に聞き流し、コーヒーを飲んでメンタル（SAN値）を回復します。時には無視も重要です。

👿 大先生の攻撃パターン

大先生は以下のような「会議あるある」攻撃であなたの精神を削ってきます。

論点のすり替え: 痛いところを突かれると、全く別の話をし始めます。

責任転嫁: 「君が決めたことだよね？」とハシゴを外してきます。

横文字連発: 「アグリー」「フィックス」「ゼロベース」等を多用し、混乱させてきます。

ちゃぶ台返し: 決定事項を直前でひっくり返します。精神的ダメージ大。

武勇伝: 「僕が若かった頃は…」という、今役に立たない昔話で時間を奪います。
